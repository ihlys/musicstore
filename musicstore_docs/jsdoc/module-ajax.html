(static) fetch(url, prototypingMode) → {object}

Creates a promise that sends XMLHttpRequest on specified URL address. Also adds X-Requested-With header with value "XMLHttpRequest" to request.
Parameters:
Name Type Description
url string the URL address of request to send
prototypingMode boolean determines if prototyping mode is enabled. If enabled then request response is always resolved regardles of status code.
